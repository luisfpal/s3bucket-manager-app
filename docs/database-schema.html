Bucket Explorer Database Schema

Django stores application metadata, tenant activation state, sharing records, file upload records, and admin configuration. Ceph RGW stores objects. RGWSquared owns bucket lifecycle and upstream permission state. In the buckets table, name is the generated bare bucket name; RGWSquared maps it to the tenant-prefixed physical bucket in Ceph.
